# Getting Started

To start contributing to psychoroid.com, follow these steps:

1. Fork the psychoroid.com repository on GitHub.

2. Clone your forked repository to your local machine.
   ```bash
   git clone https://github.com/lomiafrica/psychoroid.com
   ```
3. Create a new branch for your feature, bug fix, or improvement.
   ```bash
   git checkout -b my-feature-branch
   ```
4. Make your changes and commit them with descriptive messages.
   ```bash
   git commit -m "Add new feature: my-feature-update"
   ```
5. Push your changes to your forked repository.
   ```bash
   git push origin my-feature-branch
   ```
6. Open a pull request on the main psychoroid.com repository, describing your changes and referencing any relevant issues.

Before submitting a pull request, please ensure that your code follows our contributing guidelines and passes all tests.

If you have any questions or need assistance, feel free to reach out to our team at [hello@psychoroid.comafrica](mailto:hello@psychoroid.comafrica) or open an issue on GitHub. We plan to launch a Discord server soon for more interactive discussions!
