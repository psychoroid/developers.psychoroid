# Bounty Program

To encourage and reward contributions to lomi., we have established a bounty program. Developers can earn rewards for completing specific tasks, such as:

- Developing new payment integrations
- Implementing new features
- Solving technical issues and bugs
- Improving the platform's security
- Enhancing documentation and tutorials

## Available Bounties

Check out our [Bounties Board](https://github.com/lomiafrica/lomi/issues?q=is%3Aissue+is%3Aopen+label%3Abounty) on GitHub to see the list of available bounties and their respective rewards.

## Claiming a Bounty

To claim a bounty, follow these steps:

1. Choose a bounty from the [Bounties Board](https://github.com/lomiafrica/lomi/issues?q=is%3Aissue+is%3Aopen+label%3Abounty) that you want to work on.
2. Comment on the issue, expressing your interest in taking on the bounty.
3. The lomi. team will review your request and assign the bounty to you if approved.
4. Work on the bounty, following the contributing guidelines and any specific requirements mentioned in the issue.
5. Submit a pull request with your solution, referencing the bounty issue.
6. The lomi. team will review your pull request and merge it if it meets the requirements.
7. Once your pull request is merged, you will receive the bounty reward.

## Bounty Rewards

Bounty rewards are paid out in USD via PayPal or other agreed-upon payment methods. The reward amount for each bounty is specified on the [Bounties Board](https://github.com/lomiafrica/lomi/issues?q=is%3Aissue+is%3Aopen+label%3Abounty).

We reserve the right to modify the bounty program and reward amounts at any time.

Happy bounty hunting!
